# Executável do Projeto 🚀

Este diretório contém a versão compilada e standalone (autônoma) da **Agenda de Contatos**, gerada a partir do script Python original utilizando a ferramenta `auto-py-to-exe`.

## 📦 O que é este arquivo?
- É um binário/executável que roda diretamente no terminal.
- **Não requer instalação do Python:** Todas as dependências e o interpretador necessário foram empacotados junto com o programa.

## 🖥️ Como Executar

### No Windows:
Abra o prompt de comando (CMD) ou PowerShell na pasta e execute:
```cmd
.\agenda.exe
```
### No Linux/ macOS:
Abra o terminal na pasta e execute:
```cmd
chmod +x agenda
./agenda
