# Diretório de Banco de Dados 🗄️

Este diretório é reservado exclusivamente para armazenar o arquivo do banco de dados local do projeto (`agenda.db`).

## 📁 Como funciona?
- O arquivo do banco de dados **não é enviado para o repositório** para evitar conflitos de dados pessoais e manter o repositório limpo.
- Ao executar o script principal do projeto (`agenda.py`) pela primeira vez, o SQLite3 se encarregará de criar o arquivo `agenda.db` e todas as tabelas necessárias automaticamente dentro desta pasta.

---
*Nota: Este arquivo README também serve para garantir que o Git rastreie e mantenha a estrutura desta pasta no repositório, mesmo que ela esteja inicialmente vazia.*
